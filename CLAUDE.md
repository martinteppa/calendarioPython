## Git

After every code change, automatically commit and push using:

```bash
git add <changed files>
git commit --author="Martin Teppa <martinteppa@gmail.com>" -m "commit message"
git push origin main
```

- Always use `--author="Martin Teppa <martinteppa@gmail.com>"` on every commit.
- Always push to `origin main` after committing.
- No need to ask for confirmation before committing or pushing.
